Repository των ερωτημάτων 5 και 6 της ενότητας l3594-c1-u2.

Ο κώδικας έχει εκτελεστεί τοπικά σε Visual Studio Code με έκδοση Python 3.11.9. Δε μπορούσε να εκτελεστεί στο Google Colab, καθώς η εκεί έκδοση Python 3.12 δεν είναι συμβατή με το pycaret (supports Python 3.9-3.11).

To directory images περιέχει screenshots από όσα διαγράμματα (plots) δεν παρουσίασαν σφάλμα κατά την εκτέλεσή τους.
